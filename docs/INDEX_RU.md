# INDEX (RU): вход для нового чата/ИИ

Актуально на: 2026-02-04 12:00 UTC

Если ты новый ИИ/чат для этого проекта:
1) Прочитай **`docs/WORKFLOW_CONTRACT_RU.md`** и следуй ему строго.
2) Дальше выбирай файл по задаче:
   - Продукт и границы: `docs/PRD_RU.md`
   - Технические контракты: `docs/TECH_SPEC_RU.md`
   - Эксплуатация, деплой, перенос, инциденты: `docs/OPS_RUNBOOK_RU.md`
3) “Почему всё так устроено” и принципы документации: `docs/DOCS_RATIONALE_RU.md`
4) План работ: `docs/ROADMAP_RU.md`
5) Решения (когда нужен “почему так”): `docs/adr/`
6) История изменений релизно: `CHANGELOG.md`

## Source of Truth
- **Правила работы (чат/архив/команды/пути)**: `docs/WORKFLOW_CONTRACT_RU.md` (главный документ)
- **Контракты системы (API/данные/инварианты)**: `docs/TECH_SPEC_RU.md`
- **Операционные процедуры (runbook)**: `docs/OPS_RUNBOOK_RU.md`
- **Продуктовые границы и DoD**: `docs/PRD_RU.md`
- **История релизов/важных изменений**: `CHANGELOG.md`

## Сценарии (как пользоваться)
- Новый чат/ИИ → CONTRACT → по задаче PRD/TECH/OPS.
- Инцидент → OPS: “90 секунд” → rollback → постмортем.
- Перенос VPS → OPS: Migration TL;DR.
- Изменение публичного поведения/контрактов → TECH + CHANGELOG (+ ADR если меняется решение/инвариант).

## Правило дисциплины файлов
- Не создавай новые файлы, если можно обновить существующие.
- Если новый файл всё же нужен: обнови этот INDEX и добавь причину в историю изменений соответствующего документа.
